# Lokito — лендинг (дизайн-спека)

**Дата:** 2026-06-25
**Тип:** Одностраничный лендинг (landing page)
**Стек:** Чистый HTML / CSS / JS, без сборщика

## 1. Контекст и цель

**Lokito (ლოკიტო)** — компания в Грузии, продаёт умные замки и смежные
системы, с полным циклом услуг: от заказа до доставки, установки и настройки.
По запросу привозит любой товар под заказ.

**Цель лендинга:** показать каталог продуктов и дать посетителю удобно
связаться несколькими способами. Главное действие — изучить товары и
заказать/позвонить.

**Источник контента:** презентация `_E1_83_9A...PDF` (берётся за основу,
допускается доработка текстов).

## 2. Ключевые решения

| Параметр | Решение |
|----------|---------|
| Языки | Грузинский (по умолчанию), Русский, Английский — переключатель в шапке |
| Стиль | Тёмный премиум-тех: navy-градиент, золотые акценты, мягкое свечение |
| Контакт | Телефоны (кликабельные) + WhatsApp + Viber. Telegram не используем |
| Форма | Поля имя+телефон+товар; кнопка собирает сообщение и открывает WhatsApp/Viber. Бэкенд не нужен |
| Фото товаров | Извлекаются из PDF в `img/` |

## 3. Структура файлов

```
ლოკიტო/
├── index.html

└── img/            (логотип + фото товаров из PDF)
```

## 4. Дизайн-система

- **Цвета:** тёмно-синий фон (#0b1f2e → #0e2d44 градиент), золото (#d4af37 /
  #e8c66a), белый/светло-серый текст, мягкие тени и glow под карточками.
- **Типографика:** крупные заголовки, читаемый body. Шрифт с поддержкой
  грузинского, кириллицы и латиницы (системный стек или один веб-шрифт,
  встроенный/локальный — без внешних запросов, если позже рендерим в Artifact).
- **Сетка:** контейнер max-width ~1200px, карточки в адаптивной grid.
- **Анимации:** плавное появление секций при скролле, hover-эффекты на
  карточках и кнопках.

## 5. Секции страницы

1. **Шапка (header/nav)** — логотип Lokito, якорные ссылки (Каталог, Способы
   открытия, Как работаем, Контакты), переключатель языка (KA/RU/EN), кнопка
   «Позвонить».
2. **Hero** — заголовок «ჭკვიანი საკეტები და სხვა სისტემები», подзаголовок,
   две кнопки: «Каталог» (скролл) и «Позвонить». Крупное фото замка.
3. **Преимущества** — 3 пункта из презентации: лучшая цена · полный сервис от
   заказа до установки и настройки · привезём под заказ любой товар.
4. **Каталог** (центральная секция) — карточки товаров:
   - **TAICHEN T2302** — дверной замок. 5 способов открытия. Цвета:
     чёрный / серебро / золото (переключение меняет фото). 450 ₾ + установка 50 ₾.
   - **TAICHEN XG02** — навесной замок, до 50 отпечатков. 100 ₾.
   - **TAICHEN G02** — замок для стеклянной двери. 100 ₾ + установка 50 ₾.
   - **Gateway G3 / G3P** — шлюз для удалённого управления (G3P с PoE).
     80 ₾ / 100 ₾.
   - На каждой карточке: фото, название, ключевые характеристики, цена,
     кнопка «Заказать» → открывает форму/мессенджер с предзаполненным товаром.
5. **5 способов открытия** — визуальный блок с иконками: сенсорный пинкод ·
   отпечаток пальца · приложение · бесконтактная карта · механический ключ.
6. **Как работаем** — шаги: заказ → доставка → установка → настройка.
7. **Контакты** — кликабельные телефоны (+995 598 33 43 80, +995 599 71 97 40),
   кнопки WhatsApp и Viber, форма заявки (имя + телефон + выбор товара),
   которая собирает текст и открывает выбранный мессенджер.
8. **Футер** — логотип, контакты, год.

## 6. Поведение (JS, `main.js`)

- **i18n:** словари для KA/RU/EN; перевод по `data-i18n`-атрибутам; выбор языка
  в `localStorage`; KA по умолчанию.
- **Мобильное меню:** бургер-тоггл.
- **Выбор цвета T2302:** клик по цвету меняет изображение/подпись.
- **Плавный скролл** по якорям из навигации и кнопок.
- **Анимация появления** секций (IntersectionObserver).
- **Форма → мессенджер:** собирает строку вида «Заявка: {имя}, {телефон},
  товар: {товар}» и открывает `wa.me`/`viber://` ссылку.

## 7. Вне области (YAGNI)

- Нет бэкенда, БД, серверной обработки форм.
- Нет корзины/онлайн-оплаты.
- Нет CMS — контент в разметке/словарях.
- Telegram-кнопок нет.

## 8. Критерии готовности

- Страница открывается локально без ошибок в консоли.
- Все три языка переключаются, тексты переведены, выбор запоминается.
- Каталог показывает 4 продукта с корректными ценами и фото из PDF.
- Кнопки «Заказать», телефоны, WhatsApp, Viber и форма ведут в правильные
  каналы с предзаполненным сообщением.
- Адаптив: корректно выглядит на мобильном и десктопе.
